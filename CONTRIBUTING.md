Contributing to TFIB Waitlist Plugin

Thanks for your interest in contributing! Small notes:

- Keep changes focused and add unit or integration tests when possible.
- Follow the PHP coding standards used in the project.
- Open a PR against `main` and include a descriptive title and changelog entry.
- For changes affecting user-facing controls, include screenshots or short gif demos.

Local dev
---------
1. Copy plugin to WP dev site `wp-content/plugins/`.
2. Run the site and activate plugin.
3. Use a local IDE and run PHP linter before committing.
